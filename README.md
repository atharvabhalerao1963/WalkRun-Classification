# 🏃 Walking vs Running Classification

## 📌 Overview
This project classifies human activity (walking vs running) using sensor data.

## 🚀 Models Used
- Logistic Regression
- Decision Tree
- Random Forest
- SVM
- XGBoost (Final Model)

## 🏆 Final Model
Tuned XGBoost (Accuracy: 99.27%)
